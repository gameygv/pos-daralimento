# Solution Vision: pos-daralimento

> Punto de Venta para DAR Alimento — carne orgánica de cerdo, pollo y borrego

## Identity

### Description

POS DAR Alimento es un sistema de punto de venta web diseñado para la empresa DAR Alimento, que comercializa carne orgánica de cerdo, pollo y borrego. El sistema cubre el ciclo completo de operación: catálogo de productos con variantes (cortes, pesos), ventas con múltiples métodos de pago, control de inventario por almacén, gestión de clientes con crédito, cortes de caja diarios, reportes de ventas y un programa de lealtad. Construido sobre React 19 + Supabase, se despliega como SPA en Vercel y opera contra un stack Supabase self-hosted en EasyPanel.

## Outcomes

| **Outcome** | **Description** |
|-------------|-----------------|
| **Venta ágil** | Interfaz de POS rápida con búsqueda, escaneo de código de barras/QR, carrito y cobro en menos de 30 segundos por transacción |
| **Control de inventario** | Seguimiento de stock por variante y almacén, con movimientos auditados (entrada, salida, ajuste) y kardex |
| **Gestión financiera** | Cortes de caja, control de créditos, cuentas por cobrar, gastos operativos y reportes de rentabilidad |
| **Multi-caja** | Soporte para múltiples cajas registradoras con folios independientes y sesiones de apertura/cierre |
| **Fidelización** | Programa de lealtad con acumulación de puntos por compra y canje |
